# CChecker

 
## Nasıl Kurulur? 

Projeyi kurmak için NodeJS ihtiyacınız var bunu sitesinden veya linux kullanıyor iseniz kütüphanelerden yükliyebilirsiniz.

```bash 
  git clone https://github.com/liiuqa/CChecker
  cd FamilyChecker

  npm install
  npm run start
```
